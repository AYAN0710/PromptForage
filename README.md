PromptForge is a basic platform that helps users compare different prompting techniques for the same input.
It generates Zero-Shot, **Few-Shot**, and **Chain-of-Thought** prompts, sends them to an LLM, and displays the responses for comparison.

  Features : 
  
- Generate Zero-Shot prompts
- Generate Few-Shot prompts
- Generate Chain-of-Thought prompts
- Compare AI responses side by side
- Save prompt history in SQLite
- REST API built with FastAPI
- Input validation using Pydantic

  TechStack:
- Python
- FastAPI
- Pydantic
- SQLAlchemy
- SQLite
- Google Gemini API
